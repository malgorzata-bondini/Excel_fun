## How to run (aka: let the macros live)

### 1) Download and unblock (Windows)
Windows sometimes flags downloaded `.xlsm` like it’s a villain origin story.

1. Right click the `.xlsm` file -> **Properties**
2. Tick **Unblock** -> **Apply**

### 2) Enable macros in Excel
1. Open the file
2. Click **Enable Editing** (if shown)
3. Click **Enable Content** (macros)

If Excel is being dramatic:
Excel -> **File** -> **Options** -> **Trust Center** -> **Trust Center Settings** -> **Macro Settings**  

It’s not you. It’s Excel.

## Safety note (boring but true)
Macros can be evil. Enable them only for files you trust.  
This repo is clean, but keep the good habits.
